PLUGIN_EVERSOLO_SCREEN_CONTROL
	EN	Eversolo Screen Control

PLUGIN_EVERSOLO_SCREEN_CONTROL_DESC
	EN	Controls the Eversolo DMP-A8 screen based on playback state for this player. When enabled, the screen turns ON when music starts playing and turns OFF after a configurable delay when playback pauses or stops. Configure the settings below for each player individually.

PLUGIN_EVERSOLO_ENABLED
	EN	Enable Eversolo Screen Control

PLUGIN_EVERSOLO_ENABLED_DESC
	EN	Tick to activate screen control for this player. Leave unticked for players that are not connected to an Eversolo.

PLUGIN_EVERSOLO_AUTO_IP
	EN	Auto-detect Eversolo IP

PLUGIN_EVERSOLO_AUTO_IP_DESC
	EN	When ticked, the plugin uses this player's live IP address to reach the Eversolo (recommended). This means the plugin automatically follows IP changes from DHCP. Untick only if the Eversolo is at a different address from the player.

PLUGIN_EVERSOLO_IP_DETECTED
	EN	Current player IP:

PLUGIN_EVERSOLO_IP
	EN	Manual Eversolo IP Address

PLUGIN_EVERSOLO_IP_DESC
	EN	Only used when auto-detect is off. Enter the Eversolo's IP address manually.

PLUGIN_EVERSOLO_PORT
	EN	Eversolo API Port

PLUGIN_EVERSOLO_PORT_DESC
	EN	HTTP control port (default: 9529). Only change if your device uses a non-standard port.

PLUGIN_EVERSOLO_DELAY
	EN	Screen Off Delay (seconds)

PLUGIN_EVERSOLO_DELAY_DESC
	EN	Seconds to wait after pause/stop before turning the screen off (default: 30, range: 0–600).
